# 超级天气课程表 - 项目合规性与隐私声明

## 一、项目合规性声明

### 1.1 主题合法性
本项目是一个面向学生用户的课程表管理与天气预报集成应用，主题积极健康，符合相关法律法规要求，不包含任何违法违规内容。

### 1.2 版权说明
- 所有代码为原创开发
- 使用开源框架 Flutter，遵循其开源协议
- 第三方 API 服务遵守其使用条款

### 1.3 API 使用规范
- 使用 OpenWeatherMap 公开免费 API 服务
- API 调用频率合理，不进行恶意调用
- 仅获取天气相关公共数据，不涉及个人隐私

---

## 二、数据来源说明

### 2.1 天气数据
- **来源**: OpenWeatherMap API (https://openweathermap.org/)
- **类型**: 公开天气数据（气温、湿度、风力、气压等）
- **使用方式**: 通过 HTTP 请求获取实时天气和预报
- **API 密钥**: 使用免费 API 密钥，符合服务条款

### 2.2 课程数据
- **来源**: 用户手动输入或 CSV 文件导入
- **存储**: 本地 SQLite 数据库存储
- **用途**: 课程表展示与提醒
- **处理**: 完全本地化，不上传任何服务器

### 2.3 设置数据
- **来源**: 用户在应用内的设置操作
- **存储**: 本地 SharedPreferences
- **内容**: 主题选择、默认城市等偏好设置

---

## 三、用户隐私处理

### 3.1 不收集的信息
本应用**不收集**以下任何用户信息：
- ? 个人身份信息（姓名、手机号、邮箱等）
- ? 设备信息（IMEI、MAC地址等）
- ? 位置信息（不使用 GPS）
- ? 用户行为数据（不进行埋点或跟踪）
- ? 任何其他个人敏感信息

### 3.2 数据存储方式
所有数据**完全本地存储**：
- 课程数据: SQLite 数据库（本地文件）
- 设置数据: SharedPreferences（本地配置）
- 无云端同步，无服务器上传

### 3.3 第三方服务
应用仅使用以下第三方服务：
- OpenWeatherMap API: 仅获取公开天气数据
- 无广告 SDK
- 无分析 SDK
- 无推送 SDK

### 3.4 数据安全
- 所有数据存储在应用沙盒内
- 不与其他应用共享数据
- 用户可随时清除应用数据
